Smart Restaurant Recommendation and Crowd Prediction System

Author: Rammanoshankar P

Abstract

This project recommends restaurants based on location and predicts crowd level using Machine Learning.

Problem Statement

Users often waste time waiting in crowded restaurants. The system helps choose less crowded restaurants.

Proposed Solution

A Logistic Regression model predicts crowd level using day, time and holiday data and recommends best restaurants.

Technologies Used

  • Python
  • Machine Learning
  • Flask
  • Scikit-learn

Conclusion

The system helps users make better dining decisions and reduce waiting time.
